= ShippinGo משלוחים - סנכרון אוטומטי של הזמנות ווקומרס עם כל שירותי המשלוחים ==

ShippinGo משלוחים היא הפתרון שלך לאוטומציה של תהליך המשלוחים בחנות ווקומרס. תוסף זה מאפשר לך לסנכרן את ההזמנות שלך עם מגוון רחב של חברות משלוחים, ולייעל את ניהול המשלוחים שלך.

= מאפייני התוסף =
1. **משלוח סטנדרטי (מדלת לדלת)**: ניהול פשוט של משלוחים ישירות לדלת הלקוח.
2. **משלוח הפוך**: הפשטת תהליך החזרת פריטים מהלקוחות.
3. **משלוח גובינא**: ניהול משלוחים מורכבים הכוללים תשלומים מהלקוחות.
4. **משלוח כפול**: ניהול משלוחים מורכבים עם מספר יעדים.
5. **יצירת משלוחים בכמות גדולה**: יצירת מספר משלוחים בקלות ובמהירות.
6. **איסוף מנקודות - בחירת מיקום במפה**: מאפשר ללקוחות לבחור נקודות איסוף באמצעות מפה אינטראקטיבית (כולל תמיכה ב-Google Maps).
7. **איסוף מנקודות - בחירת מיקום מרשימה**: מציג רשימה של נקודות איסוף ללקוחות.
8. **הדפסת תוויות משלוח**: יצירה והדפסת תוויות משלוח בקלות.
9. **הדפסת תוויות משלוח בכמות גדולה**: יצירה והדפסת תוויות משלוח בכמויות בקלות.
10. **עדכוני סטטוס משלוחים**: קבלת עדכונים בזמן אמת על סטטוס המשלוחים.
11. **ביטול משלוח**: ביטול משלוחים ישירות מדשבורד ווקומרס.

= הגדרת חשבון =
זקוקים לעזרה בהתקנה? עיינו ב-[מדריך התקנת התוסף](https://www.shippingo.ai/installation).

== התקנה ==

1. העלו את קבצי התוסף לתיקיית `/wp-content/plugins/shippingo` או התקינו את התוסף ישירות ממסך התוספים של וורדפרס.
2. הפעילו את התוסף דרך התפריט 'תוספים' בוורדפרס.
3. הגדירו את הגדרות התוסף דרך WooCommerce->Settings->ShippinGo.

== שירותים חיצוניים ==

תוסף זה משתלב עם שירותים חיצוניים לצורך פעולתו. להלן פירוט השירותים הנמצאים בשימוש בתוסף:

### 1. API של פלטפורמת ShippinGo
**תיאור השירות**:  
התוסף מתחבר ל-API של פלטפורמת ShippinGo לניהול תהליכי המשלוח, כולל רישום הזמנות והפקת תוויות משלוח.

**מידע הנשלח**:  
- **יצירת חשבון**: טופס רישום לחשבון ב-ShippinGo.
- **מידע הזמנה**: כל הזמנה שנוספת או מתעדכנת מועברת לפלטפורמה.

**תנאי שליחת נתונים**:  
- הנתונים נשלחים בעת שימוש בתוסף לרישום הזמנה או להפקת תווית משלוח.

**תנאי שימוש ומדיניות פרטיות**:  
- [תנאי שימוש](https://shippingo.ai/terms)
- [מדיניות פרטיות](https://shippingo.ai/privacy)

### 2. הפקת תוויות משלוח
**תיאור השירות**:  
שירות זה משמש להפקת תוויות משלוח עבור הזמנות המנוהלות דרך פלטפורמת ShippinGo.

**מידע הנשלח**:  
- **בקשת תווית**: בעת יצירת משלוח, התוסף שולח את מזהה ההזמנה וכל מידע נדרש נוסף להפקת תווית.

**תנאי שליחת נתונים**:  
- הנתונים נשלחים בכל פעם שמבקשים תווית משלוח.

**תנאי שימוש ומדיניות פרטיות**:  
- [תנאי שימוש](https://shippingo.ai/terms)
- [מדיניות פרטיות](https://shippingo.ai/privacy)

### מידע נוסף
- כל הבקשות ל-API של פלטפורמת ShippinGo נעשות בפרוטוקול HTTPS מאובטח לשמירה על הגנת המידע.


== שאלות נפוצות ==

לפרטים נוספים, בקרו במדור השאלות הנפוצות.

== צילומי מסך ==

1. **מסך הגדרות התוסף** - הגדרה קלה של ShippinGo מדף הגדרות התוסף.
2. **סנכרון הזמנות בפעולה** - סנכרון אוטומטי של הזמנות עם חברות משלוחים.
3. **יצירת משלוחים בכמות גדולה** - יצירת משלוחים בכמויות בקלות.
4. **בחירת מיקום לאיסוף באמצעות מפה** - בחירת מיקום איסוף מועדף באמצעות מפה אינטראקטיבית.
5. **הדפסת תוויות משלוח** - הפקת תוויות משלוח ישירות מדשבורד ווקומרס.
